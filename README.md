# Manual QA Portfolio — Alison Kate Lachica

This repository contains manual QA test artifacts I created while 
testing SauceDemo (saucedemo.com), a practice e-commerce web application.

## About Me

I have a background in full stack web development and I'm now 
building hands-on experience in QA and test automation. Understanding 
how applications are built helps me write better test cases and 
communicate more effectively with development teams.

My current focus is on:
- Manual testing and QA documentation
- API testing with Postman and Newman
- Test automation with Playwright
- CI/CD pipelines with GitHub Actions
